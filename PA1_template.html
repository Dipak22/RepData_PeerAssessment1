(Solution) Are there differences in x patterns between weekdays and weekends?

  1. Below Code creates a new factor variable in the dataset with two levels – “weekday” and “weekend” indicating whether a given date is a weekday or weekend day.
data_type <- function(date) {
    if (weekdays(as.Date(date)) %in% c("Saturday", "Sunday")) {
        "weekend"
    } else {
        "weekday"
    }
}
x$data_type <- as.factor(sapply(x$date, data_type))
  1. Below Code makes a panel plot containing a time series plot (i.e. type = "l") of the 5-minute interval (x-axis) and the average number of steps taken, averaged across all weekday days or weekend days (y-axis).
library(lattice)

steps.type <- aggregate(steps ~ interval + data_type, data = x, FUN = mean)

xyplot(steps ~ interval | data_type, data = steps.type, layout = c(1, 2), type = "l")
